Features

Built for power users who own their infrastructure

Everything you need for seamless, secure file synchronization with your own server.

Event-driven sync

Zero polling. Kernel-level file watching via ReadDirectoryChangesW. Changes detected in milliseconds.

End-to-end encryption

AES-256-GCM. Files encrypted before leaving your machine. Your VPS never sees plaintext.

Network drive support

Works on SMB, NFS, and WebDAV shares with automatic fallback to smart polling mode.

Bidirectional sync

Full two-way synchronization with conflict detection. Per-folder configuration.

Secure config

Config encrypted on disk with PBKDF2 + AES-GCM. No secrets ever written in plaintext.

Native shell overlays

Real-time overlay icons directly in Windows Explorer, just like Dropbox. Synced, syncing, or error at a glance.

Security

Security is not an afterthought

Every layer of Atlas is built with a zero-trust, defense-in-depth approach.

  • AES-256-GCM encryption for all file transfers
  • PBKDF2 with 310,000 iterations for key derivation
  • Unique initialization vector (IV) per file
  • No secrets in logs, no plaintext on disk
  • Named Pipe ACL restricted to current user
  • Zero shell string injection (ProcessBuilder only)
